# Cloud Infrastructure Monitoring and Auto-Healing System

Author: Venkatesh Nagirikanti

## Overview

This project simulates a cloud infrastructure monitoring and auto-healing system similar to real-world production environments used in AWS and large-scale distributed systems.

## Features

- Monitors CPU usage
- Monitors memory usage
- Monitors disk usage
- Detects service failures
- Simulates automatic service recovery
- Logs system health and recovery actions

## Technologies Used

- Python
- Linux
- psutil
- System Monitoring Concepts

## Use Case

This tool demonstrates system reliability engineering, infrastructure monitoring, and automation skills required for System Development Engineer roles.
